## Restaurante Italiano API

Una API REST para un restaurante italiano con endpoints para carta del menú y sistema de reservas con SQLite.

### Requisitos

- Python 3.8 o superior
- SQLite

### Instalación

1. Clona el repositorio
2. Instala las dependencias con `pip install -r requirements.txt`

### Uso

1. Inicia el servidor con `python main.py`
2. Accede a la API en `http://localhost:5000`

### Endpoints

- `GET /menu`: Obtiene la carta del menú
- `POST /reservas`: Crea una reserva
